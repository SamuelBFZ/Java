Construir un programa que dada una serie de vehiculos caracterizados por su marca, modelo y precio imprima las propiedades del vehiculo mas barato.
Se debe leer por teclado las caracteristicas de cada uno de los vehiculos y crear una clase que represente cada uno de ellos.

Clase Automovil
Atributos:
    marca : String
    modelo: String
    precio: float
Metodos:
Automovil(String marca, String modelo, Float precio)
getPrecio() : float
mostrarDatos() : String
